By using this service you grant worldwide irrevocable permission to analyze and process your image and text in order to create calendar items and a calendar file. In doing so, we do not store your image or text! Your image and text are stored in memory for your session and are not stored or logged on the server. The text and image information are sent to OpenAI for intepretation. The image is loaded only into server memory and displayed back to you using browser memory and not stored server-side. The ICS calendar file is generated on-demand and is not stored on the server. If you have any questions or concerns, please contact hello@ericsoto.net.
